# Overview

Ant is an open-source, self-hosted habit tracker that runs on your private server. It’s a web-based app focused on daily habit logging, streaks, and reminders so users can maintain routines without unnecessary complexity.

Because it’s self-hosted, data stays under individual control — no external accounts, third-party trackers, or vendor lock-in. Storage and backup can be configured as desired, preserving full ownership of activity and metrics.
