# Requirements

Hardware and software requirements for Ant:

## Hardware

- OS: Linux (recommended) or other Unix-like 64-bit OS. Non-Linux hosts (Windows/macOS) can work with Docker Desktop or WSL2 but may reduce troubleshooting options.
- RAM: Minimum 3 GB; 6 GB recommended for smoother operation.
- CPU: Minimum 2 cores, recommended 4 cores.

## Software

- Docker Engine: CLI for Linux servers.
- Docker Desktop: GUI app for Windows and macOS.
