# Docker

Docker Compose is the recommended way to run Ant.

## Set up

### Step 1 - Download files

Create a folder to hold the compose file and environment variables, then change into it.

```bash title="Move to the directory"
mkdir ./ant-app
cd ./ant-app
```

Download the docker-compose file and the example environment file into the folder.

```bash title="Download files"
wget -O docker-compose.yml https://github.com/martinvasquez1/habit-tracker/releases/latest/download/docker-compose.yml
wget -O .env https://github.com/martinvasquez1/habit-tracker/releases/latest/download/default.env.example 
```

### Step 2 - Populate the .env 

Edit .env to set values:

```bash title="Default content"
VERSION=v0.1.0

# Web

VITE_API_URL="http://localhost:2002"

# API

PORT=3001
DB_URL=postgres://postgres:postgres@db:5432/ant-db
ORIGIN="http://localhost:2001"
APP_URL=http://localhost:2002
JWT_SECRET="123"
CREATE_ADMIN_PASSWORD=123
UPLOAD_DIR=./uploads
```

### Step 3 - Start

Execute the compose command to launch Ant.

```bash title="Start the app"
docker compose up -d
```

## Try

The web application will be available at http://YOUR_SERVER_IP:2001 once the setup is complete. Open this address in your browser to access and start using the application.
