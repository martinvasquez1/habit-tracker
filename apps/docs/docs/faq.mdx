# FAQ

## What is Ant?

Ant is an open-source, self-hosted habit tracker that runs on your own server, letting you log habits, track streaks, and export data.

## How is my data handled?

All data is stored on the server you run Ant on — there are no external accounts or third‑party trackers. Backups, exports, and storage location are controlled by you.

## Which deployment method is supported?

Docker Compose is the only supported deployment method for Ant.

## Can I run Ant on Windows or macOS?

Yes, just use Docker Desktop. Linux with Docker Engine is preferred.

## Where should I ask for help or report bugs?

Open issues in the project repository and include logs and environment details
